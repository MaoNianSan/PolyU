# stagev2 AD classifier experiment report

## Purpose
This run evaluates early, middle, late, fused, and sequential-interaction classifiers. Early BM25 features represent Cookie Theft information-unit capture; middle BGE-M3 features represent semantic integration; late qwen3-235b-a22b expressive-form scores represent connected-speech production quality after content masking.

## Confirmed experimental policy
- Main label order: `[disease, early, middle, late]`.
- MMSE thresholds: early/mild = 21-24, middle/moderate = 13-20, late/severe = <=12.
- Raw data are the only input. Early, middle, and late features are regenerated by this project.
- Historical outputs, historical features, old cache files, and old model files are not reused.
- Models are ranked and selected by held-out external accuracy.
- CV and OOF predictions are internal diagnostics, not the final selection rule.
- The external set is treated as held-out external validation, not as an unbiased final test after model selection.
- RBF SVM is included as a nonlinear baseline.
- MMSE is used only for severity grouping, not as an input feature.
